End-to-End Medical Chatbot (LLaMA 2)

A production-ready RAG chatbot that answers medical questions by retrieving from an indexed document corpus — grounded, accurate, and fully deployable.

LLaMA 2LangChainPineconeFlaskPython 3.9RAG

RAG Pipeline

Documents are ingested, chunked, and embedded into Pinecone vector DB. At query time, LangChain retrieves the top-k semantically similar chunks and passes them as context to LLaMA 2.

Why LLaMA 2

Open-weights LLM allows on-premise deployment, critical for HIPAA-sensitive medical data. Fine-tuned on medical Q&A pairs to improve domain accuracy.

Results

Relevance score improved +18% over vanilla LLM baseline. Flask API serves the chatbot as a web interface with streaming responses.

Deep Learning · Vision · NLP  ·  2023

Image Captioning — Attention Encoder-Decoder

An end-to-end vision-language model that generates natural language descriptions of images using a CNN encoder with LSTM + attention decoder.

PyTorchtorchvisionNLTKCNN+LSTMBahdanau AttentionFlickr8k

Encoder

A pre-trained ResNet CNN encodes input images into a fixed-length feature vector capturing semantic visual information across spatial regions.

Decoder with Attention

A Bahdanau attention-based LSTM decoder generates captions word-by-word, attending to relevant image regions at each step — producing spatially-grounded, accurate descriptions.

Training & Evaluation

Trained on Flickr8k with BLEU-1 through BLEU-4 evaluation metrics. Teacher forcing during training; beam search at inference for improved caption quality.

Movie Recommendation System

A three-tier recommendation engine on MovieLens combining popularity-based, collaborative filtering, and content-based approaches.

Scikit-learnPythonTF-IDFCosine SimilarityCollaborative FilteringMovieLens

Popularity-Based

Handles cold-start by recommending globally trending films, weighted by both average rating and number of ratings to avoid popularity bias.

Collaborative Filtering

User-user and item-item similarity matrices using cosine similarity on rating vectors. Identifies users with similar taste to recommend unseen films.

Content-Based

TF-IDF vectorization on movie genres, descriptions, and metadata. Cosine similarity recommends films with similar content profiles to what a user has rated highly.

MCQ Generator (LangChain)

Automated multiple-choice question generator from any text — configurable by topic, difficulty, and count. Powered by LangChain chains and OpenAI GPT.

LangChainOpenAI APIPythonNLPPrompt Engineering

How It Works

User provides a text passage + parameters (topic, question count, difficulty). LangChain chains orchestrate: question generation → distractor creation → answer validation in sequence.

Prompt Engineering

Custom few-shot prompts ensure well-formed MCQs with plausible distractors and unambiguous correct answers. Chain-of-thought prompting improves question quality.

Use Cases

Ideal for educators, online course creators, and quiz platforms needing scalable content generation. Produces formatted question sets ready for import into LMS systems.
